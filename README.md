A front-end Expense Tracker showcasing responsive UI design, DOM manipulation, local storage, and JavaScript fundamentals 

## 🚀 Features

- Add income and expense transactions
- Automatically calculate total balance
- View total income and total expenses
- Delete transactions
- Responsive and user-friendly interface
- Data saved using Local Storage

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)


## ▶️ How to Run

1. Clone this repository.
2. Open the project folder.
3. Open `index.html` in your web browser.
